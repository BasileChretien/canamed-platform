CANAMED — Facilitator Quick-Start

Run a clinical-communication session end-to-end. No technical setup — it works in any modern browser.

  1. Create the session
    • Open the platform and choose Create a session.
    • Enter your name, a workshop label, a scenario, and a facilitator password.
    • You land on the admin dashboard and get a 6-character session code.
  2. Get students in
    • Share the session code (or the QR code on the dashboard). Students just open the site and type the code — no account needed.
    • They give a display name + a one-tap consent, then wait in the lobby. Watch the waiting-room count fill.
  3. Start & run the modules
    • Press Start — students are auto-placed into balanced rooms.
    • Module A — clinical reasoning: the team works the chart, forms hypotheses, and votes on the key decisions.
    • Module B — breaking bad news: students pick roles (physician / patient / family / observer), run the SPIKES roleplay, then swap roles and replay from the other side.
    • Use Advance all rooms → to move the cohort through the stages together.
  4. Watch the room (no clicking into each one)
    • The dashboard shows, per room: stage + time-in-stage, live participation/equity, and who is not yet contributing — so you can nudge by name.
    • The session pacing strip at the top flags rooms calling for help, rooms over their planned time, and quiet rooms — your whole-session triage at a glance.
  5. Close & report
    • Press End session to close it and download the full archive.
    • Then generate, in one click each:
      • 📊 Impact report — a dean-ready summary (participation, equity, decision quality).
      • 📋 Accreditation evidence — competencies exercised, mapped to your framework.
      • 🎓 Attestations — printable certificates for the students.
      • 🔬 Research export — analysis-ready, pseudonymous JSON.
      • 📈 Program overview — your cumulative reach + trend across sessions.
Tips. A co-facilitator can join the same dashboard with the code + password. Debrief by discussing the physician's choices, never the person. Remind students they're assessed on their reasoning, not their English — silence and simple words are fine.

Rehearse first (optional)

Create a throw-away session and join it yourself in a second browser tab to walk the whole flow before teaching. You can end and re-create sessions freely — there's no cost or limit for normal workshop sizes.
